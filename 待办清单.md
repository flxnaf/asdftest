3. **填入临床团队资料**
   - 文件：`src/i18n/dict.ts`
   - 搜索并替换以下方括号（中英文各一）：
     - `[Clinical Director name]` / `[臨床總監姓名]`
     - `[Head of Treatment Planning]` / `[治療規劃主管姓名]`
     - `[Senior Consultant]` / `[資深顧問姓名]`
   - 每位需填：姓名、职位、学历资历、简介、英文首字母缩写

> 注意：网站正式显示的中文使用繁体（香港），上述方括号位于繁体字典内，请保持繁体输入。

## 开发与构建

```bash
git clone https://github.com/flxnaf/asdftest.git
cd asdftest
npm install
npm run dev      # 本地预览 http://localhost:5173
npm run build    # 生产构建，输出至 dist/
```

构建产物为静态文件（`dist/` 目录），按照贵司既有的静态托管流程上线即可。
