## 開發與部署

```bash
git clone https://github.com/flxnaf/asdftest.git
cd asdftest
npm install
npm run dev      # 本機預覽 http://localhost:5173
npm run build    # 生產建置（部署前驗證）
```

**Vercel：** 直接 connect repo 即可，會自動偵測為 Vite 專案，無需額外配置。每次 push 到 `main` 自動部署。
